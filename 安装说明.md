# Vue Auto I18n - 安装说明

## 📦 安装包

**vue-auto-i18n-2.0.0.tgz** (82KB) - 统一安装包，包含所有功能

---

## 🚀 安装方法

### 1. 复制文件到你的项目

将 `vue-auto-i18n-2.0.0.tgz` 复制到你的项目根目录

### 2. 使用包管理器安装

选择你使用的包管理器运行：

**pnpm (推荐):**
```bash
pnpm add ./vue-auto-i18n-2.0.0.tgz
```

**npm:**
```bash
npm install ./vue-auto-i18n-2.0.0.tgz
```

**yarn:**
```bash
yarn add ./vue-auto-i18n-2.0.0.tgz
```

**bun:**
```bash
bun add ./vue-auto-i18n-2.0.0.tgz
```

### 3. 完成！一条命令搞定

安装后会自动包含所有子包功能，无需额外安装

---

## 📚 使用示例

### Vite 插件配置

```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoI18n from 'vue-auto-i18n'

export default defineConfig({
  plugins: [
    vue(),
    AutoI18n({
      translator: {
        type: 'google',
        apiKey: process.env.GOOGLE_TRANSLATE_API_KEY
      },
      locales: ['en', 'ja', 'ko'],
      entry: 'src/main.ts'
    })
  ]
})
```

### 使用 CLI 工具

安装后直接使用命令：

```bash
# 提取并翻译中文
vue-auto-i18n extract --translate

# 只提取不翻译
vue-auto-i18n extract

# 替换代码中的中文为 i18n 调用
vue-auto-i18n replace
```

### 导入子模块

```typescript
// 核心功能
import { extractChinese, translateText } from 'vue-auto-i18n/core'

// 替换工具
import { replaceChineseInCode } from 'vue-auto-i18n/replacer'

// Vite 插件
import AutoI18n from 'vue-auto-i18n/vite'
```

- 推荐使用 pnpm，速度更快

## ❓ 常见问题

**Q: 安装失败怎么办？**
A: 确保使用 Node.js >= 18.0.0，尝试清除缓存：
```bash
pnpm store prune
rm -rf node_modules
pnpm add ./vue-auto-i18n-2.0.0.tgz
```

**Q: 如何更新到新版本？**
A: 下载新版本的 .tgz 文件，重新运行安装命令即可

**Q: 支持哪些框架？**
A: 支持 Vue 3 + Vite 的项目
